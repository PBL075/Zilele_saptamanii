Zilele_saptamanii
Un program simplu în C# care utilizează instrucțiunea switch pentru a gestiona zilele săptămânii.

Ce face programul:
Cere utilizatorului să introducă un număr de la 1 la 7.

Folosește structura switch pentru a potrivi numărul cu ziua corespunzătoare (ex: 1 = Luni, 7 = Duminică).

Afișează un mesaj de eroare prin cazul default dacă numărul nu este valid.

Concepte C# aplicate:
Structura Switch: Pentru o ramificare clară a codului.

Citirea din consolă: Utilizarea Console.ReadLine() și conversia tipurilor de date.

Validare: Gestionarea input-ului incorect.

Cum se rulează:
Asigură-te că ai .NET 8.0 instalat.

Rulează comanda dotnet run în terminalul proiectului.
